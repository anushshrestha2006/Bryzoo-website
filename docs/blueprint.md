# **App Name**: Bryzoo

## Core Features:

- Route Selection: Allow users to select their origin (Birgunj or Kathmandu) and destination.
- Date and Time Selection: Implement a datepicker for ride date selection. Add optional time slots.
- Vehicle Type Selection: Allow users to choose the type of vehicle (Sumo, EV, etc.).
- Interactive Seat Map: Display a visual seat map showing available, booked, and reserved seats.
- Booking Confirmation: Provide booking confirmation with a unique ID and ride details.
- Admin Login: Implement a login system for administrators (super admins and sub-admins) to manage the system.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) for reliability and travel.
- Background color: Light grayish-blue (#E6F7FF) for a clean, airy feel.
- Accent color: Electric blue (#7DF9FF) to highlight interactive elements and calls to action.
- Headline font: 'Poppins', sans-serif, for a contemporary, precise feel.
- Body font: 'PT Sans', sans-serif, to ensure readability in blocks of text.
- Use consistent and clear icons to represent vehicle types and booking status.
- Mobile-first responsive design to ensure accessibility across devices.
